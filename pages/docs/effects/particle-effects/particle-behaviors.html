Particle Behaviors

Particle behaviors modify particles after they have been spawned. Each behavior performs a specific function:

  • Bounds Behavior - Confines particles to a specific area
  • Color Gradient Behavior - Changes particle color over time
  • Fade Out Behavior - Gradually fades out particle alpha
  • Flies Behavior - Creates erratic flying patterns
  • Gravity Behavior - Applies gravity to particles
  • Move Behavior - Moves particles along an axis
  • Opacity Behavior - Changes particle opacity over lifetime
  • Pull Along Behavior - Pulls particles along with moving effect
  • Raycast Behavior - Detects collisions using raycasts
  • Size Curve Behavior - Changes particle size over lifetime
  • Velocity Behavior - Affects particle velocity with friction
  • Wind Behavior - Applies wind forces to particles
